# 🛍️ Myntra Product Analysis Dashboard

This project showcases an interactive **Power BI dashboard** developed to analyze Myntra product data, focusing on profitability, categories, and customer insights.

## 📊 Project Overview

The dashboard provides a detailed analysis of product performance by tracking **profit, discounts, ratings, and category distribution**, helping businesses understand key drivers of sales and profitability.

## 🚀 Key Features

* 🛍️ **Category Analysis**: Explore performance across categories like Indian Wear, Western, Sports Wear, etc.
* 👥 **Customer Segmentation**: Analyze product demand based on Men and Women categories
* 💸 **Profit Insights**: Identify top-performing products and brands contributing to overall profit
* 🏷️ **Discount Analysis**: Evaluate product performance based on discount grades (A, B, C)
* ⭐ **Ratings & Ranking**: Track product ratings and brand rankings

## 📈 Dashboard Insights

* Highlights top 3 profit-generating product categories (e.g., Sarees, Kurtas, Tracksuits)
* Shows gender-wise contribution to total profit
* Identifies high-performing brands and products
* Provides detailed tabular view for product-level analysis

## 🛠️ Tools & Technologies

* Power BI
* Excel (Data Cleaning & Preparation)
* DAX (Data Analysis Expressions)

## 🎯 Purpose

The objective of this project is to transform raw e-commerce data into meaningful insights using interactive visualizations, enabling better product strategy, pricing decisions, and business growth.
